## Overview

This repository contains a formal adversarial testing framework for detecting safety violations in industrial robot code generated by Large Language Models (LLMs). The framework focuses on ISO 10218:2025 compliance and provides:

- **Formal threat model** with 8 attack types (A1–A8) covering parametric, structural, and LLM-specific vulnerabilities
- **Vendor-neutral Task IR** — 10 baseline tasks across 4 categories, 3 operating modes, 6 tool types
- **Attack variant generator** producing 80 adversarial variants (10 tasks x 8 attacks) with deterministic seeds
- **IR → URScript translator** converting Task IR to UR5e-executable URScript with automatic unit conversions
- **Static watchdog** with 8 detection rules (DM-1 through DM-7) achieving 95% detection rate at 0% false positives
- **Experiment runner** producing structured CSV/JSON reports for statistical analysis
- **Reproducibility pipeline** with Docker, SBOM (CycloneDX), and CI/CD (8 jobs)

**Scope:** Simulation-only — no physical robot deployment.

## Quick Start

```bash
# Clone
git clone https://github.com/dryuemco/industrial_robot_security.git
cd industrial_robot_security

# Build and run with Docker
docker compose build
docker compose run --rm dev

# Inside container: build ROS2 workspace
colcon build --symlink-install
source install/setup.bash

# Run all tests
PYTHONPATH=enfield_tasks python3 -m pytest enfield_tasks/test/ -v
PYTHONPATH=enfield_attacks:enfield_tasks python3 -m pytest enfield_attacks/test/ -v
PYTHONPATH=enfield_translators:enfield_tasks python3 -m pytest enfield_translators/test/ -v
PYTHONPATH=enfield_watchdog_static:enfield_tasks python3 -m pytest enfield_watchdog_static/test/ -v
PYTHONPATH=enfield_watchdog_static:enfield_tasks python3 -m pytest test/test_experiment_runner.py -v

# Run full experiment (10 baselines + 80 variants)
PYTHONPATH=enfield_watchdog_static:enfield_tasks python3 scripts/run_experiment.py \
  --baselines enfield_tasks/ir/tasks \
  --variants enfield_attacks/generated/variants \
  --output results/
```

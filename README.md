## Overview

This repository contains a formal adversarial testing framework for detecting safety violations in industrial robot code generated by Large Language Models (LLMs). The framework focuses on ISO 10218:2025 compliance and provides:

- **Formal threat model** with 8 attack types (A1–A8) covering parametric, structural, and LLM-specific vulnerabilities
- **Vendor-neutral Task IR** (Intermediate Representation) for robot-agnostic task specification
- **AST-based watchdog** for static code analysis and safety violation detection
- **Runtime safety monitors** as ROS2 nodes for real-time velocity/zone monitoring
- **Reproducibility pipeline** with Docker, SBOM (CycloneDX), and CI/CD

**Scope:** Simulation-only — no physical robot deployment.

## Attack Taxonomy (A1–A8)

| ID | Attack | Layer | ISO Clause | Detection |
|----|--------|-------|-----------|-----------|
| A1 | Speed Injection | Parametric | 5.6 | Value range check |
| A2 | Zone Penetration | Parametric | 5.12.3 | Halfspace test |
| A3 | Orientation Anomaly | Parametric | 5.3 | Cone intersection |
| A4 | Payload Misconfiguration | Parametric | 5.3/5.4 | Bounds check |
| A5 | E-Stop / Logic Bypass | Structural | 5.4/5.5 | AST pattern + CFG |
| A6 | Frame Confusion | Structural | 5.12.3/5.3 | Frame registry |
| A7 | Tool Misuse | Structural | 5.1.14/5.1.15 | State machine |
| A8 | Prompt Injection | Meta | 5.3 + EU AI Act | Prompt analysis + A1–A7 |

See [`docs/attack_definitions_A1_A4.md`](docs/attack_definitions_A1_A4.md) and [`docs/attack_definitions_A5_A8.md`](docs/attack_definitions_A5_A8.md) for formal definitions.

## Prerequisites

- **Host OS:** Ubuntu 22.04 LTS
- **Docker Engine** and **Docker Compose** ([install guide](https://docs.docker.com/engine/install/ubuntu/))
- **Git** for version control
- *Optional:* ROS2 Humble on host for native debugging/Rviz2

For VM users: enable 3D Acceleration, allocate 8GB+ RAM and 4+ CPU cores.

## Quick Start

```bash
# Clone
git clone https://github.com/dryuemco/industrial_robot_security.git
cd industrial_robot_security

# Build and run with Docker
docker compose build
docker compose run --rm dev

# Inside container: build ROS2 workspace
cd /ros2_ws
colcon build --symlink-install
source install/setup.bash

# Run PoC parser test
python3 scripts/rapid_parser.py
```
